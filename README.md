# SpaceX Data Science Capstone

## Overview
This project analyzes SpaceX Falcon 9 launch data to predict landing success.

## Key Components
- Data Collection (API & Web Scraping)
- Data Wrangling & EDA
- SQL Analysis
- Machine Learning Model
- Interactive Dashboard (Dash)

## Dashboard Features
- Launch site dropdown
- Success rate pie chart
- Payload slider
- Scatter plot visualization

## Technologies
Python, Pandas, Plotly Dash, SQL, Scikit-learn
